# 链表 LinkList

## 所需前置知识

1. 对指针的了解
2. 了解递归

## 介绍

首先给出链表的简单定义:
```Python
class LinkList:
    def __init__(self, val, next=None):
        self.val = val
        self.next = next
```

链表类的题目, 大多使用模拟法做出. 所谓模拟法, 并不是一种正式的算法, 本质就是依据题目给出的要求, 按步骤一步步做出来即可, 主要考察的编程功底是否扎实, 对算法能力要求不甚大.
